### ✨ Key Features

* **Fully Automated & Intelligent Scanning:** Automatically tests and pings official Cloudflare IPv4 and IPv6 ranges with no manual configuration needed.
* **Modern Dark Mode UI:** Beautiful, smooth, dark-themed interface built with PySide6 (Qt).
* **Simultaneous Bilingual System:** Full support for **Persian** and **English** with instant RTL/LTR layout switching.
* **Real Download Speed Test:** Measures clean IP download speed in megabytes per second.
* **Regional (IATA) Filtering:** Filter and test IPs by specific Cloudflare data centers (e.g., SJC, FRA, etc.).
* **Easy Export:** Extract final clean IP list as a `.txt` file with one click.
* **Fully Asynchronous (Asyncio):** Ultra-fast scanning thanks to non-blocking architecture.

### 🚀 Local Setup

If you prefer to run the project directly from source code, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AbadanVpn/Cloudflare-Scanner.git
   cd Cloudflare-Scanner
   ```

2. **Install dependencies:**
   This project requires Python 3.8 or higher.
   ```bash
   pip install PySide6 aiohttp
   ```

3. **Run the application:**
   ```bash
   python main.py
   ```
